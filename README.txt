IbtikarZ - Creative Engineering Studio Landing Page
This is a high-performance, luxury-themed landing page developed for IbtikarZ, a creative engineering startup. The project focuses on a sleek "Charcoal Dark" aesthetic to represent high-tech innovation and spatial web design.

🚀 Key Features & Enhancements
Luxury Custom Theme: Extensively modified the base template to implement a bespoke "Charcoal Luxury" dark aesthetic.

Interactive Engagement: Engineered and integrated a custom floating WhatsApp chatbot button for real-time client communication.

Mobile Optimized: Ensured full responsiveness across all devices, from high-performance workstations like the HP ZBook to mobile screens.

Performance Engineering: Optimized asset delivery and integrated secure contact triggers via Formspree.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Sass), JavaScript.

Icons & UI: FontAwesome 6.0 and custom SVG integrations.

Deployment: Managed via GitHub Pages with custom domain configuration.
